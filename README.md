# Topocentric-Astro-Auto

## Description
**Topocentric-Astro-Auto** is a project designed to automate the calculations of topocentric astrology for research purposes. The tool simplifies and streamlines the computation of astrological aspects based on topocentric coordinates. It currently supports automated calculations for Precision Solar Returns (PSSR), Transits, Primary Directions (PDs), and is in development to include automated Secondary Progressions.

## Key Features
- Automated calculation of Precision Solar Returns (PSSR).
- Automated computation of Transits.
- Automated Primary Directions (PDs) calculations.
- Ongoing development to include automated Secondary Progressions.
- Utilizes Swiss Ephemeris for high-precision astronomical data.
- Previous versions of the code used Astro Seek data for calculations.
